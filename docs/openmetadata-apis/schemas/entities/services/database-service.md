## Definitions
**_databaseServiceType_**

 - Type of database service such as MySQL, BigQuery, Snowflake, Redshift, Postgres...
 - Type: `string`
 - The value is restricted to the following: 
	 1. _"BigQuery"_
	 2. _"MySQL"_
	 3. _"Redshift"_
	 4. _"Snowflake"_
	 5. _"Postgres"_
	 6. _"MSSQL"_
	 7. _"Hive"_
